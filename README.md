## 🚀 How to Run a Quest

Once the application is open, the dashboard appears. Follow this process to start your automation:

1. Log into your Discord account within the provided browser window.
2. Navigate to the Quest tab in your Discord settings.
3. Accept the quest you want to finish.
4. Return to the application window.
5. Click the "Scan" button to detect available quests.
6. Select the quests you want to run from the list.
7. Click "Start" to begin the automation process.

The application uses local timing logic to mimic human interaction. This makes the activity look natural to Discord servers.

## 📊 Understanding the Dashboard

The dashboard shows real-time information about your tasks. You can view the status of each quest type:

- **Game Quests:** The tool launches the required game client in a minimized state.
- **Video Quests:** The system plays the required content at a set pace.
- **Stream Quests:** The application sends the necessary broadcast signals as if you were streaming.
- **Activity Quests:** The software interacts with Discord activities to register your participation.
- **Achievement Quests:** The tracker monitors your progress and signals completion when the task ends.

The status light turns green when a quest reaches 100% completion. You can stop the process at any time by clicking the "Stop" button.

## 🛡️ Safety and Reliability

The application uses behavioral simulation to prevent detection. It spaces out requests to follow a lognormal distribution. This approach mimics the timing patterns human users exhibit when completing manual tasks. 

None of your personal credentials are stored on external servers. All data management happens locally on your computer. The system clears its cache after every session to maintain your privacy.

## 🛠️ Troubleshooting Issues

If the application does not detect your quest, try these steps:

- Close and reopen the application.
- Refresh your Discord Quest page in the browser tab.
- Check your internet connection.
- Ensure you accepted the quest in the Discord settings menu.

If a specific task fails, look at the logs window at the bottom of the dashboard. The application provides details about why a module might have stopped.
